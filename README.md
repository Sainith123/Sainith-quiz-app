#  React Quiz App 

A Simple React Quiz App which contains the 7 MCQ’s which have 4 options in each.



### Version
1.0.0

## 📝 Usage

### Clone this Repo
```
https://github.com/Sainith123/Sainith-quiz-app.git
```
### Installation

```sh
$ npm install
```

### ✔️ Run

```sh
$ npm start
```

Then go to http://localhost:8000
